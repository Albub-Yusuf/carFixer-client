

# carfixer

Carfixer is service centric website based on car repair shop. This website is built using node, mongodb, express, firebase, react and some others react libraries like react-bootStrap, react-elastic-carousel etc.

# Features:

# Firebase Authentication

# Stripe payment gateway integration

# Admin Panel

# User Dashboard for checking their orders and give review

# Admin create, delete service, change customer order's status, check customer message or query.
